# Web Scraping: Google Image Downloader

## Overview
This mini-project demonstrates fundamental web scraping skills
used to collect image data from public web sources.

The focus is on:
- HTTP requests using custom headers
- HTML parsing with BeautifulSoup
- Binary file handling
- Responsible, small-scale scraping

## Tools Used
- Python
- requests
- BeautifulSoup (bs4)

## Scope & Limitations
- Designed for educational and demonstration purposes
- Google Images HTML structure is unstable
- Not intended for large-scale or automated data harvesting

## Use Case
Such utilities are often used as a preliminary step
in computer vision or exploratory image-based workflows.
